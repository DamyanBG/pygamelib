8bits
=====

.. versionadded:: 1.3.0


How to use?
-----------

Example::

    from pygamelib.gfx.core import Font
    from pygamelib.base import Text

    my_font = Font("8bits")
    my_text = Text("Lorem Ipsum", font=my_font)
    my_text.print_formatted()
